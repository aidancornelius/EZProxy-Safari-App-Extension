Help for EZProxy

Thank you for purchasing the EZProxy extension for Safari. This short help section provides a high level overview of the settings and details you will need to configure the extension correctly.

1. EZProxy configuration using OCLC's EZProxy (most common)

Many major universities, colleges and public institutions use OCLC's EZProxy to provide off-campus access to journals, book chapters, and more. To configure the EZProxy using this service you'll just need an access 'domain' from your institution.

Typically these configuration domains will look like:

  • access.library.unisa.edu.au
  • proxy.library.adelaide.edu.au
  • ezproxy.library.sydney.edu.au
  • ezproxy.deakin.edu.au
  • ezproxy.waikato.ac.nz
  • ezproxy.lib.cam.ac.uk:2048

Recommended settings for EZProxy services are already set and ready to go when you download the app. All you need to do is provide the domain name (i.e. like the examples above). If you think you've changed something by accident, the 'advanced' tab in the app should have neither box checked.

Some institutions may support or require SSL for proxy requests. Usually, the URL will look something like: https://ezproxy.library.institution.edu/. In these instances you'll need to turn on SSL in the advanced settings in the app. Go to the Advanced tab and check the top checkbox.This is not standard practice, but may be a technical requirement so if you're having trouble try this option.

Remember: your library proxy URL should never include the http:// or https:// or anything following the URL, i.e. /access?=

2. EZproxy configuration using OpenAthens Access Anywhere

Some institutions have begun to move to OpenAthens' services for proxying requests to journals. In some instances these co-exist with hosted EZProxy provisions, in other cases OpenAthens completely replaces EZProxy. If your institution is following the latter, you'll need to provide your institution's OpenAthens identifier and check the My library uses OpenAthens box in the Advanced tab.

OpenAthens identifiers are relatively simple and usually are just your institutions "TLD" or main domain name, for example:

  • unimelb.edu.au
  • unisa.edu.au
  • www.monash.edu

There are some libraries which use library.domain.edu as the identifier. If in doubt, search your library's help guides on off-campus access or try entering your institution's domain name. OpenAthens can also automatically resolve some domain names if you're unsure.

3. All set?

Once you're happy with the configuration you'll need to enable the extension in Safari. From main window (close this help guide) click the "Open Safari Preferences" button at the top left of the window. Safari should open and the Settings will appear, go to the "Extensions" tab and click the checkbox next to EZProxy Button. You will likely receive a notice about the app being able to "read any website", this is to enable the button's functionality. This app has been purpose built so that no data, telemetary or details about you ever leave your device. Anything you visit or do is entirely private and is not recorded by this app.

If you have any privacy concerns, you are welcome to examine the completely open source code for this app and raise any issues or security concerns there. For information, open the "About EZProxy" window from this app.
